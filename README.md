## ✨ About the Project

**Engineering Life** is a lightweight, responsive **online course platform** created especially for **engineering students**.

It enables students to:
- Learn courses in a structured way  
- Track unit-wise progress  
- Access video, audio, and text content  
- Login securely without passwords  

This project is built using **pure web technologies** and **Supabase**, focusing on simplicity, performance, and usability.

---

## 🎯 Why I Created This Application

As an engineering student, I noticed that:
- Most learning platforms are too complex  
- Beginners struggle with UI and navigation  
- Mobile optimization is often poor  
- Progress tracking is missing or confusing  

### 💡 My Vision
I wanted to build a platform that is:
- ✅ Simple & clean  
- ✅ Beginner-friendly  
- ✅ Mobile-first  
- ✅ Secure with Magic Link login  

This project helped me gain **hands-on experience** in:
- Authentication & authorization  
- Backend-as-a-Service (Supabase)  
- Database & media handling  
- Real-world project deployment

## 🚀 Features

### 🔐 Authentication
- Magic Link Login (No Password)
- Supabase Authentication

### 🧑‍🎓 Student Module
- Student Info Form
- Personalized Welcome Page
- Profile Page with Feedback & Logout

### 📚 Course System
- View available courses
- Enroll & manage courses
- Unit-wise learning:
  - ▶️ Video
  - 🎧 Audio
  - 📄 Text
- ✅ Completion tracking

### 🎨 UI & UX
- Fully responsive design
- Clean layout with animations
- Fast & smooth experience
